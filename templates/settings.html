📚 Guide rapide

🆓 Sans API

WaveAI v3 fonctionne avec des réponses intelligentes pré-configurées pour chaque agent.

🚀 Avec OpenAI

Accès à GPT-3.5 Turbo pour des réponses dynamiques et personnalisées.

🎯 Avec Anthropic

Utilisation de Claude pour des réponses nuancées et détaillées.

⚡ Fallback intelligent

Si une API échoue, WaveAI v3 bascule automatiquement sur l'autre ou sur les réponses par défaut.

  • Vous pouvez les supprimer à tout moment
